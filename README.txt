# SDPBL-2
カレンダーを表示するプログラム Ver0.5

内容物
calender.py  -- Pythonで実行するプログラム（Windows,Ubuntu対応）
japan.txt  -- 色をつける日付をまとめた読み込み用ファイル(日本の祝日)
readme.md   -- このファイル

モード説明
1年のカレンダー生成
1か月のカレンダー生成

読み込み用ファイルの書き方(1年のカレンダーモードのみ)
1行ごとに以下のように書く
月,日,予定(任意)
1,1,元日（1月1日元日）
1,2, （1月2日なし）
1,-14,成人の日(1月第2月曜日成人の日)

ただし月は1から12まで
日付は1から31までとそのほかは以下の通り
-1 ：第一日曜日　-8 ：第二火曜日　-15：第三火曜日　
-2 ：第一水曜日　-9 ：第二水曜日　-16：第三水曜日
-3 ：第一木曜日　-10：第二木曜日　-17：第三木曜日
-4 ：第一金曜日　-11：第二金曜日　-18：第三金曜日
-5 ：第一土曜日　-12：第二土曜日　-19：第三土曜日
-6 ：第一日曜日　-13：第二日曜日　-20：第三日曜日
-7 ：第一月曜日　-14：第二月曜日　-21：第三月曜日

-22：第四火曜日　-29：最終火曜日(準備中)
-23：第四水曜日　-30：最終水曜日(準備中)
-24：第四木曜日　-31：最終木曜日(準備中)
-25：第四金曜日　-32：最終金曜日(準備中)
-26：第四土曜日　-33：最終土曜日(準備中)
-27：第四日曜日　-34：最終日曜日(準備中)
-28：第四月曜日　-35：最終月曜日(準備中)
