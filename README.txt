モード説明
1.モード選択
起動直後はこのウィンドウが表示されます。
1年のカレンダー… 3.1年のカレンダーモードへ
1か月のカレンダー… 2.1か月のカレンダーモードへ
カレンダーを作る…モードを選択してこのボタンをクリックするとそのモードへ移動します。
終了する…アプリを終了します。

2.1か月のカレンダー生成
作成する年(西暦,必須)
・作成する年を西暦で入力します。（範囲 1590 <= 年 <= 3999）
作成する月(必須)
・作成する月を選択します。
土曜日青色表示
・土曜日の日付を青色で表示します。
日曜日赤色表示
・日曜日の日付を赤色で表示します。
※このモードでは予定の読み込みはできません。

3.1年のカレンダー生成
生成する年(西暦,必須)
・作成する年を西暦で入力します。
・1年を通してカレンダーを作成するため月指定はありません。
読み込むファイル
・予定の日の表示色を変えたり、予定一覧を表示する場合は「ファイルを開く」ボタンをクリックするか、
　読み込むファイルの絶対パスを入力する。
　※ファイル指定がない場合通常のカレンダーを生成します。
土曜日青色表示
・土曜日の日付を青色で表示します。
日曜日赤色表示
・日曜日の日付を赤色で表示します。
カレンダーを作る
・このボタンをクリックするとカレンダーを生成します。
読み込み用ファイル作成
・このボタンをクリックすると4.読み込みファイル生成モードに入ります。
終了する
・このアプリを終了します。

4.読み込み用ファイル作成
ここでは予定を読み込む専用のファイルを作成します。
起動直後に既存のファイルを編集しますか？と表示されます。
はい…既にファイルを作成しており、予定を追加する場合
いいえ…新規で予定ファイルを作成したい場合。

その後作成画面に入ります。
予定・月(必須)
・色を変えたい月を入力します。
予定・日(必須)
・色を変えたい日を入力します。
予定の内容
・予定の内容を入力します。
・ただし特殊な日にち（第1月曜日）などは特殊な日一覧を確認してください。
予定日の表示色
・その日を表示するときの色を入力します。
　※使用可能な色は「使用可能色確認」ボタンをクリックするか、URL（https://www.astrouw.edu.pl/~jskowron/colors-x11/rgb.html）にアクセスして確認してください。
　※色指定がない場合は日曜日と同じ色(赤色)となります。
予定確認
・ファイルに保存する予定一覧を確認できます。
　※ただし、特殊な日(1月第1月曜日)は「1月特殊日」と表示されます。
使用可能色確認
・ブラウザで色一覧サイト(https://www.astrouw.edu.pl/~jskowron/colors-x11/rgb.html)を開きます。
予定追加
・「予定・月」、「予定・日」、「予定の内容」、「予定日の表示色」の入力内容を保存する予定に追加します。
予定削除
・「予定・月」、「予定・日」、「予定の内容」、「予定日の表示色」の入力内容と同じ内容を保存する予定から検索し削除します。
保存
・予定確認で確認できる予定内容で保存します。
終了する
このモードを終了します。

5.読み込み用ファイルを手動で作成する。
保存形式はテキストファイル(.txt)です。
ファイルのエンコードはUTF-8(BOM無し)にしてください。
※Shift-JISでは動作しません。
入力方法
1,1,元日,green（1月1日元日　表示色：緑）
1,2, （1月2日なし　表示色：赤）
1,-14,成人の日,yellow (1月第2月曜日成人の日　表示色：黄色)

↓ここから書き方↓
------------------------------------------------------------------
1,1,元旦
1,5,
1,-14,成人の日
------------------------------------------------------------------
